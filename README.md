# Hito 1 - Pizzería Mamma Mía

Proyecto React con Vite para el desafío de aprendizaje **Pizzería Mamma Mía**.

## VER ONLINE
Puedes revisar el proyecto publicado en el siguiente enlace:

[https://desafiolatam.sinapsolabs.cl/react_01_introduccion/](https://desafiolatam.sinapsolabs.cl/react_01_introduccion/)

## Qué incluye

- `App.jsx` llama a `Navbar`, `Home` y `Footer`.
- `Home.jsx` llama a `Header` y a tres componentes `CardPizza`.
- `Navbar.jsx` incluye las variables `total` y `token`.
- `CardPizza.jsx` recibe `name`, `price`, `ingredients` e `img` por props.
- `Footer.jsx` incluye el texto solicitado.
- Bootstrap se usa para botones y cards.
- Font Awesome se usa para iconos.
- CSS propio con Flexbox y media queries para computador y celular.

## Instalación

```bash
npm install
npm run dev
```

Luego abre la URL que indique la terminal, normalmente:

```bash
http://localhost:5173
```
